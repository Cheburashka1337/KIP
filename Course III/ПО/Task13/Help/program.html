О программе

Данное программное средство предназначено для нахождения НОД.
Для этого применяются алгоритмы:
- Евклида поиска НОД;
- Бинарный алгоритм Евклида поиска НОД.

Программа разработана в рамках лабораторных работ по МДК.03.01 ТРПО. Программа разработана на языке программирования Visual C# в среде программирования Microsoft Visual Studio 2017.
